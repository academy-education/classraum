# 공지

학원 전체에 보내는 공지 — 일정 변경, 휴일, 중요한 안내 — 학부모와 학생에게 한꺼번에 전달합니다.

## 공지 만들기

1. **공지** 탭에서 우측 상단의 **새 공지**를 클릭하세요.
2. **제목**을 입력하세요(예: *2026 여름방학 일정*).
3. **내용**을 입력하세요.
4. **수업을 선택**해 받는 대상 지정:
   - 목록 상단의 **모두 선택**으로 모든 클래스에 전송
   - 특정 클래스에만 관련된 공지라면 해당 수업만 선택
5. **첨부파일** — 학부모에게 문서를 보내고 싶다면 어떤 형식의 파일이든 첨부 가능합니다.
6. **만들기**를 클릭하세요.

만들기를 클릭하면 선택한 모든 사람 — 해당 수업의 학부모 포함 — 에게 자동으로 앱 내 알림이 발송됩니다.

## 편집·삭제

목록에서 공지를 열고 편집·삭제 버튼을 사용하세요. 공지와 알림 기록도 함께 업데이트됩니다.
