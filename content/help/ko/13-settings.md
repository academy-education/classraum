# 설정

설정에는 두 개의 섹션이 있습니다:

- **일반 설정** — 모든 사용자에게 표시
- **구독 관리** — 매니저와 원장만 표시

## 일반 설정

```mockup
settings-panel
```

일반 설정은 모든 사용자 — 원장, 매니저, 교사, 학부모, 학생 — 가 자신의 앱 사용 환경을 조정하는 곳입니다. 프로필 정보, 알림 토글, 테마, 언어, 학원 브랜딩까지 한 화면에서 설정할 수 있으며, 저장 즉시 모든 변경 사항이 반영됩니다.

변경 가능한 항목:

- **개인 정보** — 이름, 이메일, 전화번호
- **알림 설정** — 어떤 이벤트 유형에 대해 앱 내 알림을 받을지
- **테마** — 라이트 또는 다크 모드
- **언어** — 한국어 또는 영어
- **학원 브랜딩** — 학원 로고 업로드(앱에서 Classraum 로고를 학원 사용자에게 대체)
- **계정 삭제** — 계정 영구 삭제 *(삭제 후 데이터 처리에 대해서는 [개인정보 및 데이터](/dashboard/help/privacy-and-feedback) 참조)*

### 학원 로고 업로드

1. **설정 → 학원 브랜딩**을 여세요.
2. 로고 파일을 업로드하세요. 권장 형식: 투명 배경의 PNG.
3. 로고가 학원의 인스턴스에서 즉시 Classraum 로고를 대체합니다.

## 구독 관리 (매니저 및 원장)

- 결제 수단 변경
- 플랜 업그레이드 또는 다운그레이드
- 이번 달 남은 저장 공간 확인
- 청구서 기록 검토

좌측 사이드바 하단의 **업그레이드** 버튼을 통해서도 업그레이드할 수 있습니다 — 같은 목적지이지만 더 빠른 경로입니다.
