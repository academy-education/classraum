# 수업

수업은 한 클래스의 특정 미팅입니다 — 예: "Mars, 10월 6일 월요일 9:00 AM." 클래스가 주간 여러 번 열리면 각 요일이 별도의 수업입니다.

## 빠른 방법: 클래스에 일정 추가

수업을 만드는 가장 쉬운 방법은 클래스 자체에서 시작하는 것입니다.

```mockup
add-schedule
```

1. **클래스**로 이동해 원하는 클래스의 **일정 추가**를 클릭하세요.
2. 요일과 시간을 추가하세요(예: *월·수 9:00 AM*).
3. 시스템이 해당 일정에 따라 1년치 수업을 자동 생성합니다.

## 개별 수업 편집·추가

```mockup
create-session
```

1. 클래스에 들어가 **수업 만들기**를 선택하세요(또는 기존 수업 열기).
2. 다음 항목을 입력하세요:
   - **클래스** — 어느 클래스에 속하는지
   - **날짜·시간** — 날짜, 시작·종료 시간
   - **장소** — 오프라인 또는 온라인
   - **강의실 이름/번호** — 선택사항; 학원이 강의실 번호를 사용한다면 입력
   - **대체 교사** — 매니저·원장만 가능. 담당 교사가 아프거나 자리 비울 때 사용. 교사 본인은 대체 교사를 지정할 수 없습니다.
   - **상태** — 예정, 완료, 취소. 학원마다 취소 수업을 목록에 남길지 삭제할지 결정할 수 있습니다.
   - **출결** — 클래스에 등록된 학생들이 미리 채워져 있습니다. 비어있다면 클래스에 학생을 먼저 추가하세요.
   - **과제** — 제목, 유형, 카테고리, 설명, 마감일, 첨부파일(선택)로 과제를 첨부합니다.

⚠️ **"완료" 상태에 대한 주의:** 수업을 완료로 저장하면, 클래스의 모든 학생과 학부모에게 과제 준비가 완료되었다는 자동 메시지가 발송됩니다. **과제가 실제로 업로드되기 전까지는 완료로 표시하지 마세요** — 메시지는 회수할 수 없습니다.

## 과제 카테고리

카테고리는 학원 전체에서 공유됩니다. 일찍 작은 세트를 정하고 일관되게 사용하세요 — 예: 언어 학원은 *읽기 / 듣기 / 쓰기*, 수학 학원은 *대수 / 기하 / 미적분*.

## 텍스트에서 가져오기 (AI)

수업에 과제를 추가할 때 **텍스트에서 가져오기** 옵션을 사용하면 다른 곳에 작성한 숙제를 붙여 넣어 AI가 자동으로 분류하게 할 수 있습니다.

```mockup
import-from-text
```

## 보강 수업

정규 일정 외 수업의 경우:

```mockup
add-makeup-session
```

1. **수업**으로 이동해 우측 상단의 **보강 수업 추가**를 클릭하세요.
2. 일반 수업과 동일한 항목을 입력하세요.
3. **보강 수업 추가**를 클릭해 저장하세요.

## 수업 템플릿

수업 생성 후 **템플릿으로 저장**이 표시됩니다. 비슷한 수업을 자주 만든다면 재사용하세요:

- **수업 만들기 → 템플릿 선택**으로 적용
- **관리**에서 저장된 템플릿 편집·삭제

## 수업 목록 개요

수업 페이지 상단에 세 개의 박스가 있습니다:

- **전체 수업** — 모든 수업
- **오늘의 수업** — 오늘 진행되는 것
- **예정된 수업** — 아직 완료 표시되지 않은 수업. 수업 후 상태 업데이트를 잊지 않도록 도와줍니다.
