# 과제

과제는 숙제, 프로젝트, 퀴즈, 시험 — 학생에게 부여하는 모든 작업을 포함합니다.

## 만드는 두 가지 방법

**방법 1 — 세션에서 (권장):**

1. **세션**으로 이동하세요.
2. 원하는 세션에서 **세션 편집**을 클릭하세요.
3. 안에서 과제를 추가하거나 수정하세요.

같은 세션 안에서 여러 과제를 한 번에 만들 수 있어 더 빠릅니다.

**방법 2 — 과제 탭에서:**

1. **과제** 탭으로 이동해 **과제 추가**를 클릭하세요.
2. 다음 항목을 입력하세요:
   - **세션** — 필수; 어느 세션에 속하는지 선택
   - **제목**
   - **설명**
   - **유형**(숙제, 퀴즈, 시험, 프로젝트)
   - **카테고리**(학원의 카테고리 목록에서)
   - **마감일**

## 제출물 채점

과제를 열고 **제출 업데이트**를 클릭하세요. 각 학생마다 한 줄씩 표시되며 다음을 설정할 수 있습니다:

- 상태 설정(대기, 제출, 미제출, 면제, 지연) — 키보드 S/M/N/E/O 단축키로 빠르게 이동
- 점수 입력(자동으로 상태가 "제출"로 변경됨)
- 제출일 표시
- 교사 피드백 작성(쉐브론을 클릭해 펼치면 장문 피드백 작성 가능)

일괄 단축키:

- **모두 제출로 표시** — 한 번의 클릭으로 모든 학생을 제출로 설정(이미 면제·지연인 행은 제외)
- **모든 제출에 점수** — 숫자 입력 후 적용을 클릭하면 모든 제출 행에 동일한 점수가 부여됩니다. 출석/완료 점수에 유용.
