# 과제

과제는 숙제, 프로젝트, 퀴즈, 시험 — 학생에게 부여하는 모든 작업을 포함합니다.

## 만드는 두 가지 방법

**방법 1 — 세션에서 (권장):**

```mockup
edit-session-assignments
```

1. **세션**으로 이동하세요.
2. 원하는 세션에서 **세션 편집**을 클릭하세요.
3. 안에서 과제를 추가하거나 수정하세요.

같은 세션 안에서 여러 과제를 한 번에 만들 수 있어 더 빠릅니다.

**방법 2 — 과제 탭에서:**

1. **과제** 탭으로 이동해 **과제 추가**를 클릭하세요.
2. 다음 항목을 입력하세요:
   - **세션** — 필수; 어느 세션에 속하는지 선택
   - **제목**
   - **설명**
   - **유형**(숙제, 퀴즈, 시험, 프로젝트)
   - **카테고리**(학원의 카테고리 목록에서)
   - **마감일**

## 제출물 채점

과제를 열고 **제출 업데이트**를 클릭하면 채점 모달이 열립니다 — 등록된 학생이 각각 한 줄씩 표시됩니다.

```mockup
assignment-grading
```

각 행에는 다음이 있습니다:

- **상태 배지** — 대기, 제출, 미제출, 면제, 지연. 배지 옆의 작은 키캡 글자는 키보드 단축키입니다(`S` / `M` / `E` / `O`). 행에 포커스한 뒤 해당 키를 누르면 마우스 없이도 상태가 바뀝니다.
- **점수 입력** — 0~100 사이 숫자. 점수를 입력하면 상태가 자동으로 "제출"로 바뀝니다.
- **피드백 입력** — 짧은 교사 메모. **변경사항 저장**을 클릭하면 함께 저장됩니다.

상단의 동작 영역에는 큰 반을 빠르게 처리할 때 유용한 두 가지 일괄 단축키가 있습니다:

- **모두 제출로 표시** — 한 번의 클릭으로 모든 행을 "제출"로 설정합니다(이미 면제·지연인 행은 제외).
- **모든 제출에 점수** — 숫자를 입력하고 **적용**을 누르면 모든 제출 행에 같은 점수가 부여됩니다. 참여·완료 기반 점수를 줄 때 유용합니다.

평균 점수가 푸터에 실시간으로 표시되어 작업 중에도 반 평균을 확인할 수 있습니다.
