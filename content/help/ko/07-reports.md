# 리포트

학생의 리포트(성적표) 생성 — 일반적으로 월간. AI 피드백은 선택한 기간의 교사 코멘트와 과제 점수를 자동으로 가져옵니다.

## 리포트 목록

```mockup
reports-list
```

리포트 목록에는 보기 권한이 있는 모든 리포트가 표시됩니다(매니저는 전체, 교사는 본인 것). 오른쪽의 **상태 배지**가 게시 여부를 알려줍니다.

각 행의 **점 세 개 메뉴**에는 **편집**, **미리보기**, **삭제**가 있습니다 — 초안을 게시하거나, 오타를 수정하거나, 생성한 리포트를 삭제할 때 사용합니다.

## 리포트 만들기

```mockup
add-report
```

1. **리포트**로 이동해 **리포트 추가**를 클릭하세요.
2. **학생**을 선택하고 **성적표 제목**을 입력하세요.
3. **기간**을 선택하세요(예: 월의 첫째 날부터 마지막 날까지).
4. **과목**과 **카테고리**를 선택하세요.
   - 카테고리를 하나만 선택하면 해당 카테고리만 성적표에 표시됩니다.
5. 학생이 등록된 **수업**을 선택하세요.

이 모달 하단에는 자주 사용하는 세 가지 동작이 있습니다:

- **리포트 미리보기**(왼쪽) — 아래의 미리보기 화면이 열립니다. 저장 전에 확인할 수 있습니다.
- **리포트 만들기** — **초안**으로 저장. 매니저가 게시 전 검토하도록 권장합니다.
- **만들기 및 완료**(녹색) — 즉시 저장하고 학부모와 학생에게 전송합니다.

## 미리보기 및 AI 피드백

```mockup
report-preview
```

미리보기는 가족이 실제로 받게 될 리포트 화면을 그대로 보여줍니다: 그라데이션 학생 헤더, 리포트 범위(과목 / 카테고리 / 수업), 그리고 AI 생성 피드백 블록.

**AI 피드백 생성**을 클릭하면 AI가 선택한 기간의 교사 코멘트와 점수를 가져와 건설적인 피드백과 학습 팁을 작성합니다. 필요시 AI 텍스트를 인라인으로 직접 수정할 수도 있습니다.

## 초안 나중에 게시

1. 목록에서 리포트를 찾으세요.
2. **점 세 개 메뉴** → **편집** → **업데이트 및 완료**를 클릭하세요.
3. 상태 배지가 초안에서 **완료**로 바뀌고 리포트가 전송됩니다.

## 누가 무엇을 보는가

- **매니저**는 학원 전체의 모든 성적표를 봅니다.
- **교사**는 본인이 만든 성적표만 봅니다.
- **학부모와 학생**은 본인 것만 봅니다.

## 구독 한도

구독에 따라 만들 수 있는 성적표 수가 결정됩니다. 현재 한도는 **설정 → 구독 관리**에서 확인 가능합니다.
