# 클래스룸

클래스룸은 학원의 한 강의 단위입니다 — 예: 4학년 영어 "Mars". 수업, 과제, 출석, 성적표가 모두 클래스룸에 연결됩니다.

## 클래스룸 목록

```mockup
classrooms-list
```

왼쪽 패널의 **클래스룸**을 클릭하면 학원의 모든 클래스룸이 카드 형태로 표시됩니다. 각 카드에는 담당 교사, 학년, 과목, 수강생 수, 수업 시간이 한눈에 보입니다. 상단의 통계 카드는 활성 클래스룸 수를 알려주며, 우측 상단에서 검색·필터로 빠르게 찾을 수 있습니다.

## 클래스룸 만들기

```mockup
create-classroom-form
```

1. 왼쪽 패널에서 **클래스룸**을 클릭하세요.
2. 우측 상단의 **클래스룸 만들기**를 클릭하세요.
3. 팝업되는 양식을 작성하세요:
   - **클래스룸 이름**, **학년**, **과목**, **담당 교사** — 필수 항목.
   - **색상** — 편의를 위한 항목. 한눈에 클래스룸을 구분하는 데 도움이 됩니다.
   - **수업 일정** — 요일과 시간 선택. 설정하면 1년치 수업이 자동으로 생성됩니다.
   - **학생 등록** — 학생 추가. *참고: 학생은 시스템에 먼저 등록되어 가족과 연결되어 있어야 추가할 수 있습니다.*
   - **메모** — 클래스룸과 관련된 알림 사항.
4. `*`가 표시된 항목은 필수입니다.
5. **클래스룸 만들기**를 클릭하세요. 새 클래스룸이 클래스룸 탭에 표시됩니다.

## 클래스룸 편집·삭제

클래스룸 탭에서 해당 클래스룸을 열고 편집·삭제 버튼을 사용하세요. 삭제된 클래스룸은 먼저 **보관함**으로 이동합니다 — 실수였다면 복원할 수 있습니다.
