# Assignments

Assignments cover homework, projects, quizzes, tests — anything you give students to do.

## Two ways to create

**Option 1 — From a session (recommended):**

```mockup
edit-session-assignments
```

1. Go to **Sessions**.
2. Click **Edit Session** on the session you want.
3. Add or modify assignments inside.

This is the faster path because you can create several assignments in the same session at once.

**Option 2 — From the Assignments tab:**

1. Go to **Assignments** and click **Add Assignment**.
2. Fill in:
   - **Session** — required; pick which session this assignment belongs to
   - **Title**
   - **Description**
   - **Type** (homework, quiz, test, project)
   - **Category** (from your academy's category list)
   - **Due Date**

## Grading submissions

Open the assignment, then **Update Submissions**. Each student gets a row where you can:

- Set status (Pending, Submitted, Not Submitted, Excused, Overdue) — use the S/M/N/E/O keyboard shortcuts to fly through the list
- Enter a score (auto-flips status to Submitted)
- Mark a submitted date
- Leave teacher feedback (click the chevron to expand for longer-form notes)

Bulk shortcuts:

- **Mark all submitted** — one-click set everyone to Submitted (except already-excused/overdue rows)
- **Score for all submitted** — type a number, hit Apply, every submitted row gets it. Useful for participation/completion grades.
