# Sessions

A session is one specific meeting of a class — e.g. "Mars, Monday Oct 6 at 9:00 AM." If a class meets multiple days per week, each day is its own session.

## The fast way: add a schedule to a classroom

```mockup
add-schedule
```

The easiest way to create sessions is from the classroom itself.

1. Go to **Classrooms** and click **Add Schedule** on the class you want.
2. Add the days and times (e.g. *Mon & Wed, 9:00 AM*).
3. The system automatically creates sessions for the entire year based on that schedule.

## Editing or adding a single session

```mockup
create-session
```

1. Click into the classroom and select **Create Session** (or open an existing one).
2. Fill in:
   - **Classroom** — which class this belongs to
   - **Date & Time** — date, start, end
   - **Location** — offline or online
   - **Room Name/Number** — optional, if your academy uses room IDs
   - **Substitute Teacher** — managers/owners only. Use when the regular teacher is sick or unavailable. Teachers can't assign their own substitutes.
   - **Status** — Scheduled, Completed, or Cancelled. Each academy can decide whether cancelled sessions stay in the list or get deleted.
   - **Attendance** — pre-populated with the class's enrolled students (if any). Add students to the classroom first if this is empty.
   - **Assignments** — attach assignments by filling in title, type, category, description, due date, and optional attachments.

⚠️ **About "Completed" status:** when you save a session as Completed, an automated message goes out to all students and parents in the class saying the assignments are ready. **Don't mark Completed until your assignments are actually uploaded** — the message can't be retracted.

## Assignment categories

Categories are shared across the entire academy. Pick a small set early and stick to it — e.g. *Reading / Listening / Writing* for a language school, or *Algebra / Geometry / Calculus* for math.

## Import from text (AI)

```mockup
import-from-text
```

When adding assignments to a session, the **Import from Text** option lets you paste in homework you wrote elsewhere and have the AI auto-categorize it. Drop in a block of text, and Classraum splits it into individual assignment titles with type and category guessed for you. You can edit any field before saving.

## Make-up sessions

```mockup
add-makeup-session
```

For a class running outside its normal schedule:

1. Go to **Sessions** and click **Add Make-up Session** in the top-right corner.
2. Fill in the same fields as a regular session.
3. Click **Add Make-up Session** to save.

## Session templates

After creating a session, you'll see **Save as Template**. Reuse it later for sessions with similar details:

- **Create Session → Select a Template** to apply
- **Manage** to edit or delete saved templates

## The session list overview

Three boxes at the top of the Sessions page:

- **Total Sessions** — everything
- **Today's Sessions** — what's happening today
- **Scheduled Sessions** — filters to sessions that haven't been marked Completed yet. Useful as a reminder to update statuses after class.
